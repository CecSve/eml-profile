# GBIF EML profile

The GBIF Metadata Profile XML schema. Tagged versions are released officially on [rs.gbif.org](https://rs.gbif.org/schema/eml-gbif-profile/).

For more information about the terms and concepts used to define the GBIF Metadata Profile, you can refer to the [Reference Guide](https://www.gbif.org/sites/default/files/gbif_resource/resource-80640/gbif_metadata_profile_guide_en_v1.pdf).

Please refer to this [How-to guide](https://ipt.gbif.org/manual/en/ipt/2.6/gbif-metadata-profile) for help creating a metadata document that conforms to the GBIF Metadata Profile.
